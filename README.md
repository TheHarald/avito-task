## Установка <a name = "setup"></a>

```
git clone https://github.com/TheHarald/avito-task
cd /avito-task
```
Если хотите запустить на своей локальной машине:
```
yarn install
```


## Запуск <a name = "startup"></a>
- На локальной машине:
```
yarn start
```
- В докер-контейнере:
```
docker-compose-up
```

Открыть http://localhost:3000/
